# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).


clone el repositorio git clone https://github.com/Daniela01plwqpo/frontend

Archivos:
src/components/Productos.vue: Componente que consulta al backend y muestra la lista de productos.  
App.vue: Componente raíz que renderiza `Productos.vue`.  
main.js: Inicializa la aplicación Vue.  
vite.config.js: Configuración de Vite. 

requisitos:
tener una rura de backend funcionando 
nodejs
npm 
axios
vue
vite 
HTML/CSS


cuando finalmente tengamos nuestroa backend corriendo en la terminal entramos a la carpeta FE\vite-project
